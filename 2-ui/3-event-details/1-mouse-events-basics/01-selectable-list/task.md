# 選択可能なリスト

ファイルマネージャのような、要素が選択可能なリストを作ってください。

- リスト要素でクリックすると、その要素だけを選択します(クラス `.selected` を付与)、その以外の要素は非選択にします。
- `key:Ctrl` (Mac だと `key:Cmd`) と一緒にクリックされた場合、要素上の選択は切り替わりますが、他の要素は変更されません。
