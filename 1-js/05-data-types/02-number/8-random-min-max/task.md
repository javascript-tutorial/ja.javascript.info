# min から max のランダムな数値

組み込み関数 `Math.random()` は `0` から `1` (`1` は含まない)のランダムな値を作成します。

`min` から `max` (`max` は含まない)のランダムな小数値を生成する関数 `random(min, max)` を書いてください。

動作例:

```js
alert( random(1, 5) ); // 1.2345623452
alert( random(1, 5) ); // 3.7894332423
alert( random(1, 5) ); // 4.3435234525
```
