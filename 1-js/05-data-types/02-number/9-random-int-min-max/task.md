# min から max のランダムな整数

取りうる値として `min` と `max` を含む、`min` から `max` のランダムな *整数* 値を生成する関数 `randomInteger(min, max)` を作りなさい。

範囲 `min..max` からの任意の値は、同じ確率で現れなければいけません。

動作例:

```js

alert( random(1, 5) ); // 1
alert( random(1, 5) ); // 3
alert( random(1, 5) ); // 5
```
=======
alert( randomInteger(1, 5) ); // 1
alert( randomInteger(1, 5) ); // 3
alert( randomInteger(1, 5) ); // 5
```
